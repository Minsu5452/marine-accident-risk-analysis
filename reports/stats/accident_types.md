# 사고종류 구성

정제 사고 28,935건을 사고종류(MTIS 분류)로 집계했습니다. 이 분포는 공공데이터포털(data.go.kr)에
공개된 한국해양교통안전공단(KOMSA)의 '사고종류별 통계'와 같은 성격의 집계입니다.

| 사고종류 | 건수 | 비율 |
|---|---|---|
| 기관손상 | 7,986 | 27.6% |
| 충돌 | 4,547 | 15.7% |
| 부유물감김 | 3,349 | 11.6% |
| 침수 | 1,993 | 6.9% |
| 추진축계손상 | 1,717 | 5.9% |
| 안전사고 | 1,567 | 5.4% |
| 좌초 | 1,479 | 5.1% |
| 화재 ∙ 폭발 | 1,243 | 4.3% |
| 운항저해 | 1,206 | 4.2% |
| 조타장치손상 | 991 | 3.4% |
| 전복 | 792 | 2.7% |
| 해양오염 | 652 | 2.3% |
| 기타 | 593 | 2.0% |
| 침몰 | 440 | 1.5% |
| 접촉 | 380 | 1.3% |

- 기계·비기상형 소계(기관손상·부유물감김·추진축계손상·조타장치손상·운항저해): **15,249건 (52.7%)**
- 기상 민감형 소계(충돌·좌초·전복·접촉·침몰·침수): 9,631건 (33.3%)

설비 고장형(기관손상·부유물감김 등)이 가장 큰 비중을 차지합니다. 이 구성은 위험도 분석에서 기상
신호가 약하게 나오는 한 가지 이유이며, 시간층화 case-crossover에서 강한 기상 연관이 기계·비기상형
사고에 몰려 있는 것과 맞물립니다([case_crossover.md](case_crossover.md)).

재현: `uv run python scripts/run_stats.py`
